## 📖 About the Project

**Digital Saathi** is a modern web platform designed to provide users with easy access to digital services through a clean and responsive interface.

The application focuses on delivering a seamless user experience with intuitive navigation, fast performance, and a mobile-friendly design.

Whether accessed from desktop, tablet, or mobile devices, Digital Saathi ensures a consistent and smooth experience.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/digital-saathi.git
```

Navigate to the project directory

```bash
cd digital-saathi
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
